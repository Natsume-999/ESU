# 設計願景

## 一句話定位

> **武器即職業。你用什麼武器戰鬥，你就是什麼。**

界限 是一個拋棄傳統職業鎖定的 RPG 服務器。玩家不在創建角色時選擇職業，而是通過手中的武器實時定義自己的戰鬥身份。

## 我們要解決的問題

傳統 MMO 的"一人一職業"模型有幾個固有痛點：

- **選擇焦慮**：開局就要決定職業，新手對遊戲還沒了解就被迫做長期決定。
- **試錯代價高**：點錯天賦、選錯職業，往往要重練或花費高昂代價洗點。
- **角色僵化**：一個角色只能體驗一種玩法，想玩別的得重開小號。
- **專精與自由對立**：要麼鎖死專精，要麼平庸全能，難以兼得。

## 我們的答案

界限 用三個機制化解這些痛點：

1. **武器即職業** —— 拿起劍你就是劍士，換上弓你就是遊俠。身份由當前武器決定，切換即時生效。
2. **熟練度成長** —— 你越常用某類武器，它在你手中越強。變強來自"使用"，而非"選擇"。
3. **角色唯一檔案** —— 所有成長綁定玩家本人，跨武器繼承。換一把同類武器，熟練度不丟。

## 體驗目標

玩家在 界限 應該感受到：

| 我們想要的感覺 | 通過什麼實現 |
|----------------|--------------|
| "我可以隨便試，不怕選錯" | 零鎖定，任何武器隨時可用 |
| "我用心練的東西有回報" | 熟練度成長，專精更強 |
| "同一個號能玩出花樣" | 多流派獨立成長，自由切換 |
| "我的成長是我自己的" | 角色檔案綁定玩家，不綁物品 |

## 與傳統服務器的差異

```
傳統 RPG 服務器              界限
─────────────────           ─────────────────
創建角色 → 選職業            創建角色 → 直接開玩
職業鎖定玩法                武器決定玩法
洗點/轉職有代價             隨時換武器零代價
一號一玩法                  一號多流派
```

## 設計基調

::: tip 核心信條
**自由不等於平庸。** 我們給玩家完全的選擇自由，但通過熟練度讓"投入"始終有回報。想專精的玩家會成為某個流派的大師，想全能的玩家可以廣泛涉獵——兩種玩法都被系統尊重。
:::
