# 设计愿景

## 一句话定位

> **武器即职业。你用什么武器战斗，你就是什么。**

ESU 是一个抛弃传统职业锁定的 RPG 服务器。玩家不在创建角色时选择职业，而是通过手中的武器实时定义自己的战斗身份。

## 我们要解决的问题

传统 MMO 的"一人一职业"模型有几个固有痛点：

- **选择焦虑**：开局就要决定职业，新手对游戏还没了解就被迫做长期决定。
- **试错代价高**：点错天赋、选错职业，往往要重练或花费高昂代价洗点。
- **角色僵化**：一个角色只能体验一种玩法，想玩别的得重开小号。
- **专精与自由对立**：要么锁死专精，要么平庸全能，难以兼得。

## 我们的答案

ESU 用三个机制化解这些痛点：

1. **武器即职业** —— 拿起剑你就是剑士，换上弓你就是游侠。身份由当前武器决定，切换即时生效。
2. **熟练度成长** —— 你越常用某类武器，它在你手中越强。变强来自"使用"，而非"选择"。
3. **角色唯一档案** —— 所有成长绑定玩家本人，跨武器继承。换一把同类武器，熟练度不丢。

## 体验目标

玩家在 ESU 应该感受到：

| 我们想要的感觉 | 通过什么实现 |
|----------------|--------------|
| "我可以随便试，不怕选错" | 零锁定，任何武器随时可用 |
| "我用心练的东西有回报" | 熟练度成长，专精更强 |
| "同一个号能玩出花样" | 多流派独立成长，自由切换 |
| "我的成长是我自己的" | 角色档案绑定玩家，不绑物品 |

## 与传统服务器的差异

```
传统 RPG 服务器              ESU
─────────────────           ─────────────────
创建角色 → 选职业            创建角色 → 直接开玩
职业锁定玩法                武器决定玩法
洗点/转职有代价             随时换武器零代价
一号一玩法                  一号多流派
```

## 设计基调

::: tip 核心信条
**自由不等于平庸。** 我们给玩家完全的选择自由，但通过熟练度让"投入"始终有回报。想专精的玩家会成为某个流派的大师，想全能的玩家可以广泛涉猎——两种玩法都被系统尊重。
:::
