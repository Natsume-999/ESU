# 目標玩家

明確"為誰設計"，才能在功能取捨時有依據。

## 主要受眾

### 探索型玩家

喜歡嘗試不同玩法、不想被鎖定的玩家。界限 的"自由切換"正中其下懷。

- **訴求**：隨時試新東西，不怕選錯。
- **我們提供**：零鎖定，任何武器隨時上手。

### 專精型玩家

願意深耕某個流派、追求極致的玩家。

- **訴求**：投入有回報，專精能變得很強。
- **我們提供**：熟練度成長曲線，深耕帶來顯著強度提升。

### 收集型玩家

喜歡"練滿所有"、追求完整度的玩家。

- **訴求**：有長線目標，能逐個攻克。
- **我們提供**：多流派獨立成長，每個流派都是一條收集線。

## 玩家動機映射

| 玩家類型 | 核心動機 | 界限 的鉤子 |
|----------|----------|------------|
| 探索型 | 新鮮感、自由 | 隨時換武器換玩法 |
| 專精型 | 精通、強度 | 熟練度深耕、技能樹 |
| 收集型 | 完整度、目標 | 多流派全練、技能圖鑑 |
| 社交型 | 配合、分工 | 不同流派的團隊定位差異 |

## 玩家旅程預期

```
新手期（0-2 小時）
  嘗試幾種武器，找到順手的 → 體驗"零鎖定"的輕鬆

成長期（2-20 小時）
  專注 1-2 個流派練熟練度 → 體驗"用進廢退"的回報

成熟期（20+ 小時）
  多流派全面發展，鑽研技能搭配 → 體驗"角色唯一"的深度
```

## 不是為誰設計

明確邊界同樣重要：

- ❌ **硬核數值黨**：界限 不追求極端複雜的數值 build 計算（雖然有深度，但不以此為核心）。
- ❌ **純 PvP 競技玩家**：當前設計以 PvE 成長為主軸，PvP 平衡不是首要目標（可後續擴展）。
- ❌ **一次性體驗玩家**：系統的樂趣在於長線積累，速通型玩家可能感受不到深度。

## 設計啟示

::: tip 包容多種玩法
界限 的核心優勢是**同時容納多種玩家類型**。探索者、專精者、收集者都能在同一套系統裡找到樂趣，且互不衝突。設計新內容時，儘量讓它對多種玩家都有意義。
:::
