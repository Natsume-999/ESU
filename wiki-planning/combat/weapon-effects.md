# 武器效果設計

武器效果定義了具體武器的特殊能力。同一流派的武器共享流派技能，但可以有各自獨特的武器效果，製造差異化與收集價值。

## 武器的兩層能力

```
武器 = 流派能力（共享） + 武器效果（獨特）
        │                    │
   決定熟練度與技能      武器自身的特殊詞條
   (換同類武器繼承)      (跟隨武器，不繼承)
```

| 層 | 來源 | 是否繼承 |
|----|------|----------|
| 流派能力 | 武器所屬流派 | 是（綁定玩家檔案） |
| 武器效果 | 武器自身詞條 | 否（跟隨武器實例） |

::: tip 兩層分工
- **流派能力**體現"角色唯一"：換把同類武器，熟練度技能照舊。
- **武器效果**體現"裝備價值"：好武器有獨特詞條，值得追求。
兩者互補，既保留成長繼承，又給裝備收集留空間。
:::

## 武器效果類型

### 基礎屬性詞條

直接加數值的詞條：

| 詞條 | 效果 |
|------|------|
| 鋒利 | +攻擊力 |
| 迅捷 | +攻速 |
| 致命 | +暴擊率 |
| 吸血 | 命中回血 |
| 穿甲 | 無視部分防禦 |

### 觸發型效果

滿足條件觸發的特殊效果：

| 效果 | 觸發條件 | 效果 |
|------|----------|------|
| 灼燒 | 命中 | 目標持續燃燒傷害 |
| 冰縛 | 命中 | 減速目標 |
| 雷擊 | 暴擊 | 範圍連鎖閃電 |
| 處決 | 目標低血 | 高額斬殺傷害 |
| 反彈 | 受擊 | 反彈部分傷害 |

### 流派強化詞條

強化所屬流派機制的詞條：

| 流派 | 專屬詞條示例 |
|------|--------------|
| 劍士 | 連擊層數上限+，連擊傷害+ |
| 狂戰 | 破甲值+，低血時傷害+ |
| 遊俠 | 蓄力速度+，最大射程+ |
| 法師 | 法術範圍+，元素傷害+ |
| 刺客 | 背刺倍率+，閃避+ |

## 武器稀有度

> **設計決策**：用稀有度區分武器的詞條數量與強度，製造收集梯度。

| 稀有度 | 詞條數 | 獲取難度 |
|--------|--------|----------|
| 普通 | 0-1 | 易得 |
| 精良 | 1-2 | 常見掉落 |
| 稀有 | 2-3 | 中等挑戰 |
| 史詩 | 3-4 | 高難內容 |
| 傳說 | 4-5 + 專屬效果 | 終局/稀有 |

::: warning 稀有度不碾壓熟練度
武器稀有度提供"錦上添花"的強度，但**熟練度始終是成長主軸**。一個傳說武器在低熟練度玩家手裡，不該強過史詩武器在高熟練度玩家手裡。裝備是補充，不是替代。
:::

## 武器效果的設計原則

### 強化風格，而非改變風格

武器效果應當**強化武器所屬流派的玩法**，而非讓它變成別的流派：

- ✅ 給劍加"連擊傷害+"（強化劍的連擊風格）。
- ❌ 給劍加"遠程射擊"（變成了弓的玩法）。

### 有取捨，非純增益

高級武器效果可以帶來取捨，增加 build 深度：

```
示例："狂暴之刃"
  + 攻擊力 +30%
  - 受到傷害 +15%
  → 高風險高回報，適合激進打法
```

### 配合技能產生 combo

武器效果與流派技能可以形成搭配空間：

```
示例：劍士技能"旋風斬" + 武器詞條"灼燒"
  → 旋風斬命中範圍內所有敵人 → 全部附加灼燒
  → 範圍灼燒 combo
```

| 渠道 | 武器品質傾向 |
|------|--------------|
| 商店購買 | 普通-精良 |
| 怪物掉落 | 精良-稀有 |

## 武器效果配置（策劃參考）

武器作為物品，其效果詞條附加在武器上：

```yaml
武器ID:
  display: 武器顯示名
  class: 所屬流派           # 決定流派能力
  rarity: 稀有度
  base_attributes:          # 基礎屬性
    attack_damage: 8
  effects:                  # 武器效果詞條
    - type: lifesteal       # 吸血
      value: 0.1
    - type: burn            # 灼燒
      chance: 0.2
      duration: 3
```

## 武器設計檢查清單

- [ ] 流派歸屬明確
- [ ] 武器效果強化所屬流派風格，不越界
- [ ] 稀有度與詞條強度匹配
- [ ] 不碾壓熟練度成長主軸
- [ ] 高級武器有取捨或 combo 空間
- [ ] 獲取渠道與稀有度匹配
