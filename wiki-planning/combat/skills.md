# 技能系统

技能系统为每个流派提供专属能力，是流派差异化和成长深度的核心。

## 技能与流派绑定

- 每个流派有独立的**技能树**。
- 技能点来自该流派的等级成长，只能投入本流派技能。
- 已解锁技能跨流派**累积保存**，但**只在对应流派为主武器时可释放**。

```
剑流派技能树        弓流派技能树
  旋风斩              穿透箭
  致命一击            鹰眼
  连斩精通            箭雨
   ↑                  ↑
 用剑系技能点解锁    用弓系技能点解锁
```

## 技能类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **主动技能** | 玩家手动释放，有冷却/消耗 | 旋风斩、穿透箭 |
| **被动技能** | 满足条件自动触发 | 暴击回血、低血减伤 |
| **强化天赋** | 数值型被动，强化属性 | 攻击力+5%、暴击率+10% |

## 技能释放方式

> **设计决策**：用"手持主武器 + 交互动作"触发主动技能，避免与原版操作冲突。

| 触发方式 | 说明 | 默认 |
|----------|------|------|
| 潜行 + 右键 | 释放指定主动技能 | 启用 |
| 潜行 + 左键 | 释放另一技能 | 可选 |
| 连段检测 | 连续攻击触发连击技 | 流派相关 |

::: tip 进阶触发方案
快捷键释放（需客户端配合或物品栏方案）是进阶选项，初期用潜行+交互即可满足。
:::

## 技能点系统

```
流派升级 → 每若干级给 1 技能点 → 投入该流派技能树
```

- 技能点按流派独立累积。
- 解锁技能消耗技能点。
- 部分技能可多级强化，逐级投点提升。

## 冷却管理

- 每个技能有独立冷却时间。
- 冷却按流派独立记录（切换流派不重置，见 [主武器机制](/systems/main-weapon#技能冷却的跨流派处理)）。
- 冷却状态通过 HUD 显示。

## 技能设计维度

设计一个技能时考虑：

| 维度 | 问题 |
|------|------|
| **定位** | 它解决什么场景？（单体/群体/位移/生存） |
| **风格契合** | 它是否强化流派特色？ |
| **解锁等级** | 什么时候给玩家？（成长节奏） |
| **冷却/消耗** | 多强、多久能用一次？ |
| **combo 空间** | 能否与其它技能/武器词条搭配？ |

## 技能解锁节奏

技能应在流派成长过程中**逐步解锁**，配合 [进程设计](/gameplay/progression)：

| 等级段 | 技能解锁 |
|--------|----------|
| 1-15 | 第一个主动技能（核心手感） |
| 16-30 | 核心技能（流派特色成型） |
| 31-50 | 关键被动（机制深化） |
| 51-80 | 高级技能（强力能力） |
| 81-100 | 终极技能（流派巅峰） |

## 技能平衡原则

::: warning 技能平衡
- **无废技能**：每个技能都应有使用场景，避免"必点"和"没用"两极。
- **有取舍**：技能点有限时，鼓励玩家根据玩法取舍，而非堆满一种。
- **combo 而非碾压**：强力来自技能搭配，而非单技能数值碾压。
:::

## 技能效果配置（策划参考）

```yaml
技能ID:
  name: 技能显示名
  class: 所属流派
  type: active / passive
  unlock_level: 解锁等级
  cost: 消耗技能点
  cooldown: 冷却秒数        # 主动技能
  description: 效果描述
  # 主动技能效果链
  actions:
    - 伤害/位移/召唤/特效...
  # 被动技能效果
  effect:
    属性键: 数值
```

## 技能与其它系统的交互

```
技能系统
  ├─ 读取：流派等级（决定可解锁哪些技能）
  ├─ 读取：技能点（角色档案）
  ├─ 读取：当前激活流派（决定可释放哪些技能）
  └─ 交互：武器效果（combo 搭配）
```

具体技能示例见 [技能图鉴](/combat/skill-list)。
