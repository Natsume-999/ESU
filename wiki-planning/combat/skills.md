# 技能系統

技能系統為每個流派提供專屬能力，是流派差異化和成長深度的核心。

## 技能與流派綁定

- 每個流派有獨立的**技能樹**。
- 技能點來自該流派的等級成長，只能投入本流派技能。
- 已解鎖技能跨流派**累積保存**，但**只在對應流派為主武器時可釋放**。

```
劍流派技能樹        弓流派技能樹
  旋風斬              穿透箭
  致命一擊            鷹眼
  連斬精通            箭雨
   ↑                  ↑
 用劍系技能點解鎖    用弓系技能點解鎖
```

## 技能類型

| 類型 | 說明 | 示例 |
|------|------|------|
| **主動技能** | 玩家手動釋放，有冷卻/消耗 | 旋風斬、穿透箭 |
| **被動技能** | 滿足條件自動觸發 | 暴擊回血、低血減傷 |
| **強化天賦** | 數值型被動，強化屬性 | 攻擊力+5%、暴擊率+10% |

## 技能釋放方式

> **設計決策**：用"手持主武器 + 交互動作"觸發主動技能，避免與原版操作衝突。

| 觸發方式 | 說明 | 默認 |
|----------|------|------|
| 潛行 + 右鍵 | 釋放指定主動技能 | 啟用 |
| 潛行 + 左鍵 | 釋放另一技能 | 可選 |
| 連段檢測 | 連續攻擊觸發連擊技 | 流派相關 |

::: tip 進階觸發方案
快捷鍵釋放（需客戶端配合或物品欄方案）是進階選項，初期用潛行+交互即可滿足。
:::

## 技能點系統

```
流派升級 → 每若干級給 1 技能點 → 投入該流派技能樹
```

- 技能點按流派獨立累積。
- 解鎖技能消耗技能點。
- 部分技能可多級強化，逐級投點提升。

## 冷卻管理

- 每個技能有獨立冷卻時間。
- 冷卻按流派獨立記錄（切換流派不重置，見 [主武器機制](/systems/main-weapon#技能冷卻的跨流派處理)）。
- 冷卻狀態通過 HUD 顯示。

## 技能設計維度

設計一個技能時考慮：

| 維度 | 問題 |
|------|------|
| **定位** | 它解決什麼場景？（單體/群體/位移/生存） |
| **風格契合** | 它是否強化流派特色？ |
| **解鎖等級** | 什麼時候給玩家？（成長節奏） |
| **冷卻/消耗** | 多強、多久能用一次？ |
| **combo 空間** | 能否與其它技能/武器詞條搭配？ |

## 技能解鎖節奏

技能應在流派成長過程中**逐步解鎖**，配合 [進程設計](/gameplay/progression)：

| 等級段 | 技能解鎖 |
|--------|----------|
| 1-15 | 第一個主動技能（核心手感） |
| 16-30 | 核心技能（流派特色成型） |
| 31-50 | 關鍵被動（機制深化） |
| 51-80 | 高級技能（強力能力） |
| 81-100 | 終極技能（流派巔峰） |

## 技能平衡原則

::: warning 技能平衡
- **無廢技能**：每個技能都應有使用場景，避免"必點"和"沒用"兩極。
- **有取捨**：技能點有限時，鼓勵玩家根據玩法取捨，而非堆滿一種。
- **combo 而非碾壓**：強力來自技能搭配，而非單技能數值碾壓。
:::

## 技能效果配置（策劃參考）

```yaml
技能ID:
  name: 技能顯示名
  class: 所屬流派
  type: active / passive
  unlock_level: 解鎖等級
  cost: 消耗技能點
  cooldown: 冷卻秒數        # 主動技能
  description: 效果描述
  # 主動技能效果鏈
  actions:
    - 傷害/位移/召喚/特效...
  # 被動技能效果
  effect:
    屬性鍵: 數值
```

## 技能與其它系統的交互

```
技能系統
  ├─ 讀取：流派等級（決定可解鎖哪些技能）
  ├─ 讀取：技能點（角色檔案）
  ├─ 讀取：當前激活流派（決定可釋放哪些技能）
  └─ 交互：武器效果（combo 搭配）
```
