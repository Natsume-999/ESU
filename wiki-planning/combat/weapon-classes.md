# 武器流派

武器流派是職業系統的分類骨架。每件武器歸入一個流派，流派決定戰鬥風格、屬性傾向和技能樹。

## 流派定義

> **設計決策**：用武器的**自定義標籤**（如 `weapon-class: sword`）標記流派，而非用原版材質判斷。這樣同一原版材質能做不同流派的武器，也方便擴展自定義武器。

每個流派獨立擁有：

- 熟練度進度（獨立等級與經驗）
- 屬性成長曲線（不同流派傾向不同）
- 技能樹（專屬技能）
- 戰鬥風格定位

## 初始流派設計

> 以下是建議的初始流派，可增刪調整。

| 流派 ID | 名稱 | 風格定位 | 屬性傾向 | 戰鬥特色 |
|---------|------|----------|----------|----------|
| `sword` | 劍士 | 平衡近戰 | 攻速、暴擊 | 連擊流暢，攻守兼備 |
| `axe` | 狂戰 | 重型近戰 | 攻擊力、破甲 | 高傷爆發，無視護甲 |
| `bow` | 遊俠 | 遠程輸出 | 射程、蓄力 | 拉開距離，精準打擊 |
| `staff` | 法師 | 元素法術 | 法強、範圍 | 範圍殺傷，元素效果 |
| `dagger` | 刺客 | 敏捷暗殺 | 暴擊、閃避 | 高機動，背刺斬殺 |
| `fist` | 武者 | 徒手格鬥 | 移速、連段 | 赤手連擊，靈活機動 |

## 流派風格差異化

每個流派應有**鮮明的玩法差異**，而非只是數值不同：

### 劍士（平衡基準）

- 定位：上手簡單，攻守平衡，適合新手。
- 特色：連擊系統，連續攻擊疊加增益。

### 狂戰（爆發輸出）

- 定位：高風險高回報的重型輸出。
- 特色：攻擊慢但傷害高，破甲無視防禦。

### 遊俠（遠程風箏）

- 定位：保持距離，安全輸出。
- 特色：蓄力射擊，距離越遠傷害越高。

### 法師（範圍控制）

- 定位：群體殺傷與元素控制。
- 特色：法術有範圍效果，附帶燃燒/冰凍等。

### 刺客（機動爆發）

- 定位：高機動單體爆發。
- 特色：背刺暴擊，閃避反擊，斬殺低血目標。

### 武者（靈活連段）

- 定位：徒手也能戰鬥的格鬥流派。
- 特色：連段累積，高移速，不依賴武器。

## 流派定義配置（策劃參考）

每個流派的定義包含：

```yaml
流派ID:
  name: 流派顯示名
  icon: 圖標物品
  style: 風格描述
  # 屬性成長（每級增量）
  attributes:
    attack_damage: 每級攻擊力增量
    attack_speed: 每級攻速增量
    ...
  # 熟練度曲線參數
  exp_curve:
    base: 基礎經驗
    exponent: 曲線指數
  # 技能點獲取
  skill_points:
    per_level: 多少級給1點
```

具體屬性數值見 [屬性體系](/balance/attributes)，曲線見 [成長曲線](/balance/curves)。

## 流派平衡原則

::: warning 流派間平衡
不同流派應當**強度相近、風格各異**：
- 沒有"最強流派"，只有"最適合某場景的流派"。
- 每個流派都有擅長與不擅長的情況，形成取捨。
- 平衡目標是讓玩家"因為喜歡而選擇"，而非"因為最強而選擇"。
:::

## 擴展新流派

新增流派的檢查清單：

- [ ] 風格定位與現有流派有明顯區別
- [ ] 屬性傾向獨特（不與現有流派重複）
- [ ] 有至少 3-5 個專屬技能撐起技能樹
- [ ] 強度與現有流派平衡
- [ ] 武器載體設計（什麼武器屬於這個流派）

## 流派與武器的關係

```
流派（抽象分類）
  │
  ├─ 多件武器屬於同一流派
  │    劍流派 ← 木劍 / 鐵劍 / 鑽石劍 / 傳說之劍
  │
  └─ 武器強度不同，但共享流派的熟練度與技能
```
