# Progression 进程

进程设计描述玩家从新手到终局的**长期成长轨迹**。ESU 的进程是多线并行的——玩家可以同时推进多个流派。

## 进程总览

```
新手期          成长期              成熟期           终局期
(Lv 1-15)      (Lv 15-50)         (Lv 50-80)       (Lv 80-100)
─────────      ──────────         ──────────       ──────────
熟悉操作        专注主流派          多流派发展        极致精通
尝试多武器      解锁核心技能        钻研技能搭配      挑战终局内容
首次升级        稳定成长曲线        副流派起步        练满收集
```

## 单流派的进程节点

每个流派从 1 级到 100 级，设置关键里程碑：

| 等级段 | 阶段 | 解锁内容 |
|--------|------|----------|
| 1-15 | 入门 | 基础属性成长，第一个主动技能 |
| 16-30 | 进阶 | 核心技能解锁，属性显著提升 |
| 31-50 | 熟练 | 关键被动技能，流派特色机制 |
| 51-80 | 精通 | 高级技能，强力天赋 |
| 81-100 | 大师 | 终极技能，流派巅峰能力 |

::: tip 里程碑设计
在关键等级（如 10/25/50/75/100）放置**值得期待的解锁**，给玩家明确的短期目标。避免"升级只加数值"的枯燥感。
:::

## 多流派进程策略

玩家可以选择不同的发展策略，系统都支持：

### 专精流（深度优先）

```
剑  ██████████ Lv 100（大师）
其它 ░░░░░░░░░░ 几乎不练
```

- 把一个流派练到顶，成为该流派大师。
- 回报：单流派极强，技能全解锁。

### 双修流（平衡发展）

```
剑  ███████░░░ Lv 70
弓  ██████░░░░ Lv 60
```

- 主练两个互补流派（如近战+远程）。
- 回报：应对不同场景灵活切换。

### 全能流（广度优先）

```
剑 ████░ 弓 ███░ 法杖 ███░ 匕首 ██░ ...
```

- 每个流派都练一些，样样通。
- 回报：永远有合适的工具应对任何情况。

## 进程驱动的内容门槛

用流派等级作为内容解锁的门槛，引导进程：

| 内容 | 建议门槛 |
|------|----------|
| 初级副本 | 任一流派 Lv 10 |
| 中级副本 | 任一流派 Lv 30 |
| 高级副本 | 任一流派 Lv 50 |
| 终局挑战 | 任一流派 Lv 80+ |

::: warning 门槛用"任一流派"
内容门槛应基于**任一流派达标**，而非总和或特定流派。这样专精玩家和全能玩家都能进入，符合"零锁定"理念。
:::

## 进程的长度设计

| 目标 | 预估时长 | 说明 |
|------|----------|------|
| 单流派满级 | 中长线 | 专精玩家的主要目标 |
| 解锁单流派全技能 | 与满级同步 | 升级即逐步解锁 |
| 练满所有流派 | 超长线 | 收集型玩家的终极目标 |

进程长度应通过 [成长曲线](/balance/curves) 的参数调控，确保：前期快速反馈、中期稳定推进、后期长线追求。

## 进程与留存

```
新鲜感（换流派） + 成长感（升级解锁） + 目标感（里程碑/收集）
            ↓
        持续的游玩动力
```
