# Progression 進程

進程設計描述玩家從新手到終局的**長期成長軌跡**。界限 的進程是多線並行的——玩家可以同時推進多個流派。

## 進程總覽

```
新手期          成長期              成熟期           終局期
(Lv 1-15)      (Lv 15-50)         (Lv 50-80)       (Lv 80-100)
─────────      ──────────         ──────────       ──────────
熟悉操作        專注主流派          多流派發展        極致精通
嘗試多武器      解鎖核心技能        鑽研技能搭配      挑戰終局內容
首次升級        穩定成長曲線        副流派起步        練滿收集
```

## 單流派的進程節點

每個流派從 1 級到 100 級，設置關鍵里程碑：

| 等級段 | 階段 | 解鎖內容 |
|--------|------|----------|
| 1-15 | 入門 | 基礎屬性成長，第一個主動技能 |
| 16-30 | 進階 | 核心技能解鎖，屬性顯著提升 |
| 31-50 | 熟練 | 關鍵被動技能，流派特色機制 |
| 51-80 | 精通 | 高級技能，強力天賦 |
| 81-100 | 大師 | 終極技能，流派巔峰能力 |

::: tip 里程碑設計
在關鍵等級（如 10/25/50/75/100）放置**值得期待的解鎖**，給玩家明確的短期目標。避免"升級只加數值"的枯燥感。
:::

## 多流派進程策略

玩家可以選擇不同的發展策略，系統都支持：

### 專精流（深度優先）

```
劍  ██████████ Lv 100（大師）
其它 ░░░░░░░░░░ 幾乎不練
```

- 把一個流派練到頂，成為該流派大師。
- 回報：單流派極強，技能全解鎖。

### 雙修流（平衡發展）

```
劍  ███████░░░ Lv 70
弓  ██████░░░░ Lv 60
```

- 主練兩個互補流派（如近戰+遠程）。
- 回報：應對不同場景靈活切換。

### 全能流（廣度優先）

```
劍 ████░ 弓 ███░ 法杖 ███░ 匕首 ██░ ...
```

- 每個流派都練一些，樣樣通。
- 回報：永遠有合適的工具應對任何情況。

## 進程驅動的內容門檻

用流派等級作為內容解鎖的門檻，引導進程：

| 內容 | 建議門檻 |
|------|----------|
| 初級副本 | 任一流派 Lv 10 |
| 中級副本 | 任一流派 Lv 30 |
| 高級副本 | 任一流派 Lv 50 |
| 終局挑戰 | 任一流派 Lv 80+ |

::: warning 門檻用"任一流派"
內容門檻應基於**任一流派達標**，而非總和或特定流派。這樣專精玩家和全能玩家都能進入，符合"零鎖定"理念。
:::

## 進程的長度設計

| 目標 | 預估時長 | 說明 |
|------|----------|------|
| 單流派滿級 | 中長線 | 專精玩家的主要目標 |
| 解鎖單流派全技能 | 與滿級同步 | 升級即逐步解鎖 |
| 練滿所有流派 | 超長線 | 收集型玩家的終極目標 |

進程長度應通過 [成長曲線](/balance/curves) 的參數調控，確保：前期快速反饋、中期穩定推進、後期長線追求。

## 進程與留存

```
新鮮感（換流派） + 成長感（升級解鎖） + 目標感（里程碑/收集）
            ↓
        持續的遊玩動力
```
