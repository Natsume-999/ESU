# 終局內容

終局（Endgame）是玩家達到高等級後的持續遊玩內容。好的終局讓"滿級"不是終點，而是新玩法的起點。

## 終局玩家的狀態

到達終局期的玩家通常：

- 至少一個流派接近或達到滿級。
- 掌握了該流派的全部技能。
- 尋求新的挑戰與目標。

## 終局內容方向

### 方向一：橫向擴展（練新流派）

界限 的獨特優勢：滿級玩家天然有一條"再練一個流派"的路。

- 已滿級劍的玩家，可以從零開始練弓。
- 帶著對系統的理解，第二個流派的體驗是"熟練的新鮮"。
- **設計支持**：流派間可設計協同（見下文"流派組合"）。

### 方向二：縱向深化（挑戰內容）

為高等級玩家提供高難度挑戰：

| 內容類型 | 說明 |
|----------|------|
| 高難副本 | 需要熟練操作與技能搭配的 PvE 挑戰 |
| Boss 戰 | 機制複雜的精英戰鬥，考驗流派理解 |
| 無盡模式 | 波次遞增的生存挑戰，比拼極限 |
| 限時挑戰 | 週期刷新的特殊關卡 |

### 方向三：收集完整度

為收集型玩家提供長線目標：

- 練滿所有流派。
- 解鎖所有技能（技能圖鑑全收集）。
- 獲得稀有武器（不同流派的極品武器）。

## 流派組合玩法（終局深度）

> **設計決策**：終局可引入"流派協同"，鼓勵多流派發展，但需謹慎不破壞"切換即時"的核心。

可能的協同設計：

- **連攜技**：先用 A 流派打出某狀態，切到 B 流派觸發額外效果。
- **被動繼承**：某些高等級被動天賦可跨流派生效（少量、精選）。
- **雙武器流**：主手 + 副手分屬不同流派的進階玩法。

::: warning 協同的邊界
流派協同是"錦上添花"的終局深度，**不能成為強制要求**。核心循環（單流派玩法）必須始終完整可玩，否則違背"零鎖定"——玩家不該被迫練多個流派才能變強。
:::

## 終局獎勵設計

終局內容的獎勵應當：

| 獎勵類型 | 作用 | 注意 |
|----------|------|------|
| 稀有武器 | 提供更好的流派載體 | 不應碾壓熟練度成長 |
| 外觀/稱號 | 炫耀性獎勵，無數值 | 滿足成就感 |
| 收集進度 | 圖鑑、成就完成度 | 長線目標 |
| 特殊技能 | 終局專屬技能 | 高門檻解鎖 |

::: tip 獎勵不應破壞平衡
終局武器/裝備的強度提升應當**有限且可控**，不能讓"刷到神裝"碾壓"練熟練度"。熟練度始終是成長主軸，裝備是補充。
:::

## 終局的留存邏輯

```
滿級一個流派 ──→ 三條路同時打開：
                  ├─ 練新流派（橫向）
                  ├─ 挑戰高難內容（縱向）
                  └─ 追求收集完整（廣度）
                       ↓
                  持續的終局動力
```

## 終局內容檢查清單

- [ ] 滿級後有明確的"下一步"（不止一條路）
- [ ] 橫向擴展（新流派）體驗順暢
- [ ] 縱向挑戰有足夠難度與回報
- [ ] 收集目標清晰可追蹤
- [ ] 終局獎勵不破壞熟練度成長主軸
- [ ] 流派協同是可選深度，非強制要求
