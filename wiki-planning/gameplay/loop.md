# 核心循環

核心循環是玩家在 界限 中反覆進行的基礎遊戲行為。一個好的核心循環應當**簡單、有回報、可重複**。

## 主循環

```
        ┌──────────────────────────────────────┐
        │                                        │
        ▼                                        │
   選擇武器 ──→ 戰鬥 ──→ 積累熟練度 ──→ 變強 ────┘
   (定義身份)   (核心)    (成長)        (回報)
        │                                        
        │  隨時可換武器，切換流派                 
        └──────────────────────────────────────
```

### 循環四步

1. **選擇武器（定義身份）**
   - 玩家手持某武器，系統識別其流派，套用對應屬性與技能。
   - 這一步是"無成本"的——隨時可換。

2. **戰鬥（核心行為）**
   - 用當前主武器擊殺生物、挑戰內容。
   - 戰鬥體驗因流派而異：劍的連擊、弓的遠程、法杖的範圍。

3. **積累熟練度（成長）**
   - 有效戰鬥為當前流派積累熟練度經驗。
   - 經驗只進當前主武器流派。

4. **變強（回報）**
   - 熟練度升級 → 屬性提升 + 技能點 + 解鎖新技能。
   - 玩家感受到"我練的武器更強了"。

## 循環的節奏

| 時間尺度 | 循環內容 | 玩家感受 |
|----------|----------|----------|
| 秒級 | 攻擊、釋放技能 | 操作爽快 |
| 分鐘級 | 擊殺一波怪、漲一點經驗 | 穩步推進 |
| 小時級 | 升幾級、解鎖新技能 | 明顯成長 |
| 天/周級 | 練滿一個流派、轉練新流派 | 長線目標 |

## 多流派的循環疊加

界限 的核心循環可以**並行多條**：

```
玩家 A 的循環狀態：
  劍流派  ████████░░ Lv 42（主練）
  弓流派  ███░░░░░░░ Lv 15（副練）
  法杖    █░░░░░░░░░ Lv 3（嚐鮮）
```

玩家可以隨時切換正在推進的循環——今天練劍，明天練弓。每條循環獨立積累，互不干擾。這讓核心循環既有**深度**（單流派深耕）又有**廣度**（多流派發展）。

## 循環的驅動力

是什麼讓玩家願意一遍遍重複核心循環？

| 驅動力 | 來源 |
|--------|------|
| **即時反饋** | 每次攻擊的打擊感、技能特效 |
| **短期目標** | 下一級、下一個技能解鎖 |
| **長期目標** | 練滿流派、收集所有技能 |
| **新鮮感** | 隨時切換流派換玩法 |

## 設計要點

::: tip 讓每一步都有反饋
核心循環的每一步都應有清晰反饋：選武器時 HUD 顯示流派切換，戰鬥時有打擊感，漲經驗時有進度條變化，升級時有提示動畫。反饋密度決定循環的"爽快度"。
:::

::: warning 避免循環空轉
如果戰鬥不給有效回報（如掛機刷怪），循環就會"空轉"——玩家在重複但沒成長感。防刷機制（見 [熟練度成長](/systems/mastery)）確保循環始終有意義。
:::
