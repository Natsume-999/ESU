# 熟練度成長

熟練度是 界限 的核心成長貨幣。它是"用進廢退"支柱的直接實現：用什麼武器戰鬥，就積累什麼武器的熟練度。

## 核心規則

> 玩家用**當前主武器**進行有效戰鬥時，為**該流派**積累熟練度經驗。經驗只進當前主武器流派。

```
手持劍擊殺怪物 → 劍流派 +經驗
換成弓擊殺怪物 → 弓流派 +經驗（劍不變）
```

這天然鼓勵"想練哪個就用哪個"，且各流派獨立，互不衝突。

## 經驗來源

| 來源 | 說明 | 默認 |
|------|------|------|
| **擊殺生物** | 按生物最大生命值 × 係數給經驗 | 啟用 |
| **造成傷害** | 按造成傷害量 × 係數給經驗 | 可選 |

> **設計決策**：默認用"擊殺給經驗"。可選開啟"傷害給經驗"，好處是防止隊友搶最後一擊，壞處是需更強的防刷。

### 經驗公式示例

```
擊殺經驗 = 生物最大生命值 × 擊殺係數
傷害經驗 = 造成傷害 × 傷害係數（可選）
```

具體系數見 [成長曲線](/balance/curves)。

## 等級與回報

每個流派有獨立等級（建議 1-100）。升級帶來三類回報：

| 回報 | 說明 |
|------|------|
| **被動屬性** | 隨等級增長的攻擊力、攻速、生命等（按流派傾向不同） |
| **技能點** | 每升若干級給 1 點，用於解鎖/強化該流派技能 |
| **里程碑解鎖** | 特定等級解鎖關鍵技能或機制 |

::: tip 屬性只在激活流派時生效
手持劍時享受劍系全部屬性加成；換成弓，則切換為弓系加成。屬性跟隨當前主武器流派。
:::

## 熟練度曲線

```
等級 n 升級所需經驗 = base × (n ^ exponent)

示例（base=100, exponent=1.5）：
  Lv 1→2:   100
  Lv 10→11: ~3162
  Lv 50→51: ~35355
```

> **設計決策**：曲線參數全部可配置。目標是前期快速反饋（新手有成長感）、後期長線追求（專精有深度）。詳見 [成長曲線](/balance/curves)。

## 防刷機制

熟練度來自"有效戰鬥"。需防止掛機刷怪籠等無效刷取破壞"用進廢退"的意義。

### 遞減回報

> 對同一低級生物的重複擊殺，經驗遞減。

```
首次擊殺某怪：100% 經驗
重複擊殺同類：80% → 64% → ...（遞減）
一段時間後重置
```

### 掛機懲罰

> 玩家長時間不移動視為掛機，經驗大幅降低。

```
正常戰鬥：100% 經驗
判定掛機（如60秒不移動）：經驗 ×0.1
```

### 等級壓制

> 擊殺遠低於自己等級的生物，經驗降低，防止"虐菜刷級"。

## 防刷的平衡

::: warning 防刷不能誤傷正常玩家
防刷機制要精準打擊掛機/刷怪籠，但**不能影響正常遊玩**：
- 正常打副本、探索擊殺不該被誤判。
- 遞減應針對"同一地點重複刷同類怪"，正常移動戰鬥不觸發。
- 掛機判定閾值要合理，避免誤傷"站樁輸出"的正常戰鬥。
:::

## 熟練度的展示

通過 BetterHud 實時展示：

- 當前流派名稱與等級。
- 熟練度進度條（當前經驗 / 升級所需）。
- 升級時的提示動畫。
