# 熟练度成长

熟练度是 ESU 的核心成长货币。它是"用进废退"支柱的直接实现：用什么武器战斗，就积累什么武器的熟练度。

## 核心规则

> 玩家用**当前主武器**进行有效战斗时，为**该流派**积累熟练度经验。经验只进当前主武器流派。

```
手持剑击杀怪物 → 剑流派 +经验
换成弓击杀怪物 → 弓流派 +经验（剑不变）
```

这天然鼓励"想练哪个就用哪个"，且各流派独立，互不冲突。

## 经验来源

| 来源 | 说明 | 默认 |
|------|------|------|
| **击杀生物** | 按生物最大生命值 × 系数给经验 | 启用 |
| **造成伤害** | 按造成伤害量 × 系数给经验 | 可选 |

> **设计决策**：默认用"击杀给经验"。可选开启"伤害给经验"，好处是防止队友抢最后一击，坏处是需更强的防刷。

### 经验公式示例

```
击杀经验 = 生物最大生命值 × 击杀系数
伤害经验 = 造成伤害 × 伤害系数（可选）
```

具体系数见 [成长曲线](/balance/curves)。

## 等级与回报

每个流派有独立等级（建议 1-100）。升级带来三类回报：

| 回报 | 说明 |
|------|------|
| **被动属性** | 随等级增长的攻击力、攻速、生命等（按流派倾向不同） |
| **技能点** | 每升若干级给 1 点，用于解锁/强化该流派技能 |
| **里程碑解锁** | 特定等级解锁关键技能或机制 |

::: tip 属性只在激活流派时生效
手持剑时享受剑系全部属性加成；换成弓，则切换为弓系加成。属性跟随当前主武器流派。
:::

## 熟练度曲线

```
等级 n 升级所需经验 = base × (n ^ exponent)

示例（base=100, exponent=1.5）：
  Lv 1→2:   100
  Lv 10→11: ~3162
  Lv 50→51: ~35355
```

> **设计决策**：曲线参数全部可配置。目标是前期快速反馈（新手有成长感）、后期长线追求（专精有深度）。详见 [成长曲线](/balance/curves)。

## 防刷机制

熟练度来自"有效战斗"。需防止挂机刷怪笼等无效刷取破坏"用进废退"的意义。

### 递减回报

> 对同一低级生物的重复击杀，经验递减。

```
首次击杀某怪：100% 经验
重复击杀同类：80% → 64% → ...（递减）
一段时间后重置
```

### 挂机惩罚

> 玩家长时间不移动视为挂机，经验大幅降低。

```
正常战斗：100% 经验
判定挂机（如60秒不移动）：经验 ×0.1
```

### 等级压制

> 击杀远低于自己等级的生物，经验降低，防止"虐菜刷级"。

## 防刷的平衡

::: warning 防刷不能误伤正常玩家
防刷机制要精准打击挂机/刷怪笼，但**不能影响正常游玩**：
- 正常打副本、探索击杀不该被误判。
- 递减应针对"同一地点重复刷同类怪"，正常移动战斗不触发。
- 挂机判定阈值要合理，避免误伤"站桩输出"的正常战斗。
:::

## 熟练度的展示

通过 BetterHud 实时展示：

- 当前流派名称与等级。
- 熟练度进度条（当前经验 / 升级所需）。
- 升级时的提示动画。

## 多流派熟练度示例

```
玩家档案：
  剑   Lv 42  ████████░░  18400/22000 exp

```

每条独立积累，玩家自由选择推进哪条。
