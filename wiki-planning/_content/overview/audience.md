# 目标玩家

明确"为谁设计"，才能在功能取舍时有依据。

## 主要受众

### 探索型玩家

喜欢尝试不同玩法、不想被锁定的玩家。界限 的"自由切换"正中其下怀。

- **诉求**：随时试新东西，不怕选错。
- **我们提供**：零锁定，任何武器随时上手。

### 专精型玩家

愿意深耕某个流派、追求极致的玩家。

- **诉求**：投入有回报，专精能变得很强。
- **我们提供**：熟练度成长曲线，深耕带来显著强度提升。

### 收集型玩家

喜欢"练满所有"、追求完整度的玩家。

- **诉求**：有长线目标，能逐个攻克。
- **我们提供**：多流派独立成长，每个流派都是一条收集线。

## 玩家动机映射

| 玩家类型 | 核心动机 | 界限 的钩子 |
|----------|----------|------------|
| 探索型 | 新鲜感、自由 | 随时换武器换玩法 |
| 专精型 | 精通、强度 | 熟练度深耕、技能树 |
| 收集型 | 完整度、目标 | 多流派全练、技能图鉴 |
| 社交型 | 配合、分工 | 不同流派的团队定位差异 |

## 玩家旅程预期

```
新手期（0-2 小时）
  尝试几种武器，找到顺手的 → 体验"零锁定"的轻松

成长期（2-20 小时）
  专注 1-2 个流派练熟练度 → 体验"用进废退"的回报

成熟期（20+ 小时）
  多流派全面发展，钻研技能搭配 → 体验"角色唯一"的深度
```

## 不是为谁设计

明确边界同样重要：

- ❌ **硬核数值党**：界限 不追求极端复杂的数值 build 计算（虽然有深度，但不以此为核心）。
- ❌ **纯 PvP 竞技玩家**：当前设计以 PvE 成长为主轴，PvP 平衡不是首要目标（可后续扩展）。
- ❌ **一次性体验玩家**：系统的乐趣在于长线积累，速通型玩家可能感受不到深度。

## 设计启示

::: tip 包容多种玩法
界限 的核心优势是**同时容纳多种玩家类型**。探索者、专精者、收集者都能在同一套系统里找到乐趣，且互不冲突。设计新内容时，尽量让它对多种玩家都有意义。
:::
