# 武器流派

武器流派是职业系统的分类骨架。每件武器归入一个流派，流派决定战斗风格、属性倾向和技能树。

## 流派定义

> **设计决策**：用武器的**自定义标签**（如 `weapon-class: sword`）标记流派，而非用原版材质判断。这样同一原版材质能做不同流派的武器，也方便扩展自定义武器。

每个流派独立拥有：

- 熟练度进度（独立等级与经验）
- 属性成长曲线（不同流派倾向不同）
- 技能树（专属技能）
- 战斗风格定位

## 初始流派设计

> 以下是建议的初始流派，可增删调整。

| 流派 ID | 名称 | 风格定位 | 属性倾向 | 战斗特色 |
|---------|------|----------|----------|----------|
| `sword` | 剑士 | 平衡近战 | 攻速、暴击 | 连击流畅，攻守兼备 |
| `axe` | 狂战 | 重型近战 | 攻击力、破甲 | 高伤爆发，无视护甲 |
| `bow` | 游侠 | 远程输出 | 射程、蓄力 | 拉开距离，精准打击 |
| `staff` | 法师 | 元素法术 | 法强、范围 | 范围杀伤，元素效果 |
| `dagger` | 刺客 | 敏捷暗杀 | 暴击、闪避 | 高机动，背刺斩杀 |
| `fist` | 武者 | 徒手格斗 | 移速、连段 | 赤手连击，灵活机动 |

## 流派风格差异化

每个流派应有**鲜明的玩法差异**，而非只是数值不同：

### 剑士（平衡基准）

- 定位：上手简单，攻守平衡，适合新手。
- 特色：连击系统，连续攻击叠加增益。

### 狂战（爆发输出）

- 定位：高风险高回报的重型输出。
- 特色：攻击慢但伤害高，破甲无视防御。

### 游侠（远程风筝）

- 定位：保持距离，安全输出。
- 特色：蓄力射击，距离越远伤害越高。

### 法师（范围控制）

- 定位：群体杀伤与元素控制。
- 特色：法术有范围效果，附带燃烧/冰冻等。

### 刺客（机动爆发）

- 定位：高机动单体爆发。
- 特色：背刺暴击，闪避反击，斩杀低血目标。

### 武者（灵活连段）

- 定位：徒手也能战斗的格斗流派。
- 特色：连段累积，高移速，不依赖武器。

## 流派定义配置（策划参考）

每个流派的定义包含：

```yaml
流派ID:
  name: 流派显示名
  icon: 图标物品
  style: 风格描述
  # 属性成长（每级增量）
  attributes:
    attack_damage: 每级攻击力增量
    attack_speed: 每级攻速增量
    ...
  # 熟练度曲线参数
  exp_curve:
    base: 基础经验
    exponent: 曲线指数
  # 技能点获取
  skill_points:
    per_level: 多少级给1点
```

具体属性数值见 [属性体系](/balance/attributes)，曲线见 [成长曲线](/balance/curves)。

## 流派平衡原则

::: warning 流派间平衡
不同流派应当**强度相近、风格各异**：
- 没有"最强流派"，只有"最适合某场景的流派"。
- 每个流派都有擅长与不擅长的情况，形成取舍。
- 平衡目标是让玩家"因为喜欢而选择"，而非"因为最强而选择"。
:::

## 扩展新流派

新增流派的检查清单：

- [ ] 风格定位与现有流派有明显区别
- [ ] 属性倾向独特（不与现有流派重复）
- [ ] 有至少 3-5 个专属技能撑起技能树
- [ ] 强度与现有流派平衡
- [ ] 武器载体设计（什么武器属于这个流派）

## 流派与武器的关系

```
流派（抽象分类）
  │
  ├─ 多件武器属于同一流派
  │    剑流派 ← 木剑 / 铁剑 / 钻石剑 / 传说之剑
  │
  └─ 武器强度不同，但共享流派的熟练度与技能
```

武器的具体效果设计见 [武器效果设计](/combat/weapon-effects)。
