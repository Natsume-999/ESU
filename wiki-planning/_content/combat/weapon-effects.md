# 武器效果设计

武器效果定义了具体武器的特殊能力。同一流派的武器共享流派技能，但可以有各自独特的武器效果，制造差异化与收集价值。

## 武器的两层能力

```
武器 = 流派能力（共享） + 武器效果（独特）
        │                    │
   决定熟练度与技能      武器自身的特殊词条
   (换同类武器继承)      (跟随武器，不继承)
```

| 层 | 来源 | 是否继承 |
|----|------|----------|
| 流派能力 | 武器所属流派 | 是（绑定玩家档案） |
| 武器效果 | 武器自身词条 | 否（跟随武器实例） |

::: tip 两层分工
- **流派能力**体现"角色唯一"：换把同类武器，熟练度技能照旧。
- **武器效果**体现"装备价值"：好武器有独特词条，值得追求。
两者互补，既保留成长继承，又给装备收集留空间。
:::

## 武器效果类型

### 基础属性词条

直接加数值的词条：

| 词条 | 效果 |
|------|------|
| 锋利 | +攻击力 |
| 迅捷 | +攻速 |
| 致命 | +暴击率 |
| 吸血 | 命中回血 |
| 穿甲 | 无视部分防御 |

### 触发型效果

满足条件触发的特殊效果：

| 效果 | 触发条件 | 效果 |
|------|----------|------|
| 灼烧 | 命中 | 目标持续燃烧伤害 |
| 冰缚 | 命中 | 减速目标 |
| 雷击 | 暴击 | 范围连锁闪电 |
| 处决 | 目标低血 | 高额斩杀伤害 |
| 反弹 | 受击 | 反弹部分伤害 |

### 流派强化词条

强化所属流派机制的词条：

| 流派 | 专属词条示例 |
|------|--------------|
| 剑士 | 连击层数上限+，连击伤害+ |
| 狂战 | 破甲值+，低血时伤害+ |
| 游侠 | 蓄力速度+，最大射程+ |
| 法师 | 法术范围+，元素伤害+ |
| 刺客 | 背刺倍率+，闪避+ |

## 武器稀有度

> **设计决策**：用稀有度区分武器的词条数量与强度，制造收集梯度。

| 稀有度 | 词条数 | 获取难度 |
|--------|--------|----------|
| 普通 | 0-1 | 易得 |
| 精良 | 1-2 | 常见掉落 |
| 稀有 | 2-3 | 中等挑战 |
| 史诗 | 3-4 | 高难内容 |
| 传说 | 4-5 + 专属效果 | 终局/稀有 |

::: warning 稀有度不碾压熟练度
武器稀有度提供"锦上添花"的强度，但**熟练度始终是成长主轴**。一个传说武器在低熟练度玩家手里，不该强过史诗武器在高熟练度玩家手里。装备是补充，不是替代。
:::

## 武器效果的设计原则

### 强化风格，而非改变风格

武器效果应当**强化武器所属流派的玩法**，而非让它变成别的流派：

- ✅ 给剑加"连击伤害+"（强化剑的连击风格）。
- ❌ 给剑加"远程射击"（变成了弓的玩法）。

### 有取舍，非纯增益

高级武器效果可以带来取舍，增加 build 深度：

```
示例："狂暴之刃"
  + 攻击力 +30%
  - 受到伤害 +15%
  → 高风险高回报，适合激进打法
```

### 配合技能产生 combo

武器效果与流派技能可以形成搭配空间：

```
示例：剑士技能"旋风斩" + 武器词条"灼烧"
  → 旋风斩命中范围内所有敌人 → 全部附加灼烧
  → 范围灼烧 combo
```

## 武器获取渠道

| 渠道 | 武器品质倾向 |
|------|--------------|
| 商店购买 | 普通-精良 |
| 怪物掉落 | 精良-稀有 |
| 副本奖励 | 稀有-史诗 |
| Boss 掉落 | 史诗-传说 |
| 合成/强化 | 可提升现有武器 |

## 武器效果配置（策划参考）

武器作为物品，其效果词条附加在武器上：

```yaml
武器ID:
  display: 武器显示名
  class: 所属流派           # 决定流派能力
  rarity: 稀有度
  base_attributes:          # 基础属性
    attack_damage: 8
  effects:                  # 武器效果词条
    - type: lifesteal       # 吸血
      value: 0.1
    - type: burn            # 灼烧
      chance: 0.2
      duration: 3
```

## 武器设计检查清单

- [ ] 流派归属明确
- [ ] 武器效果强化所属流派风格，不越界
- [ ] 稀有度与词条强度匹配
- [ ] 不碾压熟练度成长主轴
- [ ] 高级武器有取舍或 combo 空间
- [ ] 获取渠道与稀有度匹配
