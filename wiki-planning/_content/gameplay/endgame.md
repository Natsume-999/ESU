# 终局内容

终局（Endgame）是玩家达到高等级后的持续游玩内容。好的终局让"满级"不是终点，而是新玩法的起点。

## 终局玩家的状态

到达终局期的玩家通常：

- 至少一个流派接近或达到满级。
- 掌握了该流派的全部技能。
- 寻求新的挑战与目标。

## 终局内容方向

### 方向一：横向扩展（练新流派）

界限 的独特优势：满级玩家天然有一条"再练一个流派"的路。

- 已满级剑的玩家，可以从零开始练弓。
- 带着对系统的理解，第二个流派的体验是"熟练的新鲜"。
- **设计支持**：流派间可设计协同（见下文"流派组合"）。

### 方向二：纵向深化（挑战内容）

为高等级玩家提供高难度挑战：

| 内容类型 | 说明 |
|----------|------|
| 高难副本 | 需要熟练操作与技能搭配的 PvE 挑战 |
| Boss 战 | 机制复杂的精英战斗，考验流派理解 |
| 无尽模式 | 波次递增的生存挑战，比拼极限 |
| 限时挑战 | 周期刷新的特殊关卡 |

### 方向三：收集完整度

为收集型玩家提供长线目标：

- 练满所有流派。
- 解锁所有技能（技能图鉴全收集）。
- 获得稀有武器（不同流派的极品武器）。

## 流派组合玩法（终局深度）

> **设计决策**：终局可引入"流派协同"，鼓励多流派发展，但需谨慎不破坏"切换即时"的核心。

可能的协同设计：

- **连携技**：先用 A 流派打出某状态，切到 B 流派触发额外效果。
- **被动继承**：某些高等级被动天赋可跨流派生效（少量、精选）。
- **双武器流**：主手 + 副手分属不同流派的进阶玩法。

::: warning 协同的边界
流派协同是"锦上添花"的终局深度，**不能成为强制要求**。核心循环（单流派玩法）必须始终完整可玩，否则违背"零锁定"——玩家不该被迫练多个流派才能变强。
:::

## 终局奖励设计

终局内容的奖励应当：

| 奖励类型 | 作用 | 注意 |
|----------|------|------|
| 稀有武器 | 提供更好的流派载体 | 不应碾压熟练度成长 |
| 外观/称号 | 炫耀性奖励，无数值 | 满足成就感 |
| 收集进度 | 图鉴、成就完成度 | 长线目标 |
| 特殊技能 | 终局专属技能 | 高门槛解锁 |

::: tip 奖励不应破坏平衡
终局武器/装备的强度提升应当**有限且可控**，不能让"刷到神装"碾压"练熟练度"。熟练度始终是成长主轴，装备是补充。
:::

## 终局的留存逻辑

```
满级一个流派 ──→ 三条路同时打开：
                  ├─ 练新流派（横向）
                  ├─ 挑战高难内容（纵向）
                  └─ 追求收集完整（广度）
                       ↓
                  持续的终局动力
```

## 终局内容检查清单

- [ ] 满级后有明确的"下一步"（不止一条路）
- [ ] 横向扩展（新流派）体验顺畅
- [ ] 纵向挑战有足够难度与回报
- [ ] 收集目标清晰可追踪
- [ ] 终局奖励不破坏熟练度成长主轴
- [ ] 流派协同是可选深度，非强制要求
