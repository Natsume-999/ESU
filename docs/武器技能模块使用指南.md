# 武器技能模块使用指南

> 模块名：`weapon-skill` ｜ 依赖：**MMOCore** + **MMOItems**

## 这个模块是做什么的

打破 MMOCore "选了职业才能放固定技能" 的死板设定，改成 **武器即职业**：

- 玩家**手持哪把武器，技能栏就自动绑定那把武器对应的技能**
- **换一把武器，技能组立刻整组切换**
- 技能释放仍然走 MMOCore 原生的按键施法（技能栏），本模块只负责"换槽位里绑的是什么技能"

举例：拿起「利刃」剑 → 技能栏变成 旋风斩 / 冲刺 / 治疗；切到「龙弓」→ 技能栏立刻变成 穿透射击 / 箭雨。

---

## 工作原理（一句话）

```
切换快捷栏 → 读取手持 MMOItems 武器的物品ID → 查配置找到对应技能列表
          → 动态绑定到 MMOCore 技能槽 1/2/3... → 玩家按原生按键释放
```

技能从 MMOCore **全局技能注册表**按 ID 查找，**不受职业限制**——这是"武器即职业"的关键。

---

## 三个必须满足的前提

> ⚠️ 不满足会出现"绑了但放不出技能"，这是 MMOCore 本身的机制限制，不是 bug。

1. **玩家要有一个 MMOCore 职业**
   可以是一个通用默认职业，不用真的让玩家"选职业"，但数据上得有职业，否则技能槽不存在。

2. **职业要定义足够数量的技能槽位（skill slots）**
   武器给 3 个技能，职业配置里就要有**至少 3 个 skill slot**。
   （在 MMOCore 的职业配置文件里配置 skill-slots）

3. **技能必须是可释放的主动技能**
   永久被动技能（isPermanent）会被 MMOCore 拒绝绑定。

## 配置步骤

### 第 1 步：准备 MMOItems 武器

在 MMOItems 里创建好你的武器，记下它的 **物品 ID**（如 `CUTLASS`）。
查物品 ID：手持武器，看 MMOItems 物品配置文件的键名。

### 第 2 步：准备 MMOCore 技能

确认你要绑定的技能在 MMOCore 里**已注册**，记下技能 ID（如 `WHIRLWIND`）。
技能 ID = MMOCore `skills` 文件夹里各技能的 ID。

### 第 3 步：给职业配技能槽

在 MMOCore 职业配置里，确保 skill-slots 数量 ≥ 单把武器的最大技能数。

### 第 4 步：配置本模块

编辑 `plugins/ESU/config.yml` 的 `weapon-skill` 段：

```yaml
weapon-skill:
  enable: true              # 打开模块
  max-skill-level: 3        # 绑定技能用的最大等级
  clear-when-unequip: true  # 收起武器时清空技能绑定
  message:
    enable: true            # 显示切换提示
    action-bar: true        # true=动作栏, false=聊天框
    equip: "&a已切换为 &e{weapon} &a技能组"
    unequip: "&7已收起武器技能"
  weapons:
    # 写法1：直接列技能（提示显示物品ID）
    CUTLASS:
      - WHIRLWIND     # 绑到技能槽 1
      - DASH          # 技能槽 2
      - HEALING       # 技能槽 3
    # 写法2：带显示名（提示显示自定义名字）
    DRAGON_BOW:
      name: "&b长弓"
      skills:
        - PIERCING_ARROW
        - ARROW_RAIN
```

### 第 5 步：重载

```
/esu reload
```

---

## 提示效果说明

| 配置 | 效果 |
|------|------|
| `message.enable: false` | 不显示任何切换提示 |
| `action-bar: true` | 提示显示在物品栏上方（动作栏），不刷屏 |
| `action-bar: false` | 提示显示在聊天框 |
| `{weapon}` | 占位符，替换为武器的显示名（写法2的 name，或物品ID）|

## 常见问题

**Q：换了武器但技能栏没变 / 放不出技能？**
依次检查三个前提：① 玩家有职业吗？② 职业的 skill-slots 够数吗？③ 技能是主动技能（非永久被动）吗？

**Q：技能 ID 填什么？**
MMOCore 已注册的技能 ID，不是 MMOItems 的技能。在 MMOCore `skills` 文件夹里查。

**Q：物品 ID 区分大小写吗？**
不区分。配置里会统一转大写匹配。

**Q：一把武器最多几个技能？**
取决于职业的 skill-slots 数量。给几个槽就能绑几个。

**Q：副手武器生效吗？**
当前只读主手。换快捷栏 / 加入服务器 / 重载时刷新。

**Q：和 MMOCore 原本的技能绑定冲突吗？**
本模块会接管技能槽 1~N 的绑定（N=武器技能数）。建议把武器技能交给本模块统一管理。
