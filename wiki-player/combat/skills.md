# 技能与释放

每个武器流派都有自己的技能。这一页讲技能怎么学、怎么放。

## 技能从哪来

```
升级 → 获得技能点 → 投入当前流派技能树 → 解锁技能
```

技能点是升级给的，花技能点解锁你想要的技能。

## 技能分三种

| 类型 | 怎么生效 | 例子 |
|------|----------|------|
| 主动技能 | 你手动释放，有冷却 | 旋风斩、火球术 |
| 被动技能 | 满足条件自动触发 | 暴击回血、低血减伤 |
| 强化天赋 | 直接加属性 | 攻击力+5%、暴击率+10% |

## 主动技能怎么放

> 默认用 **潜行 + 交互** 释放。

```
潜行（按住 Shift）+ 右键 → 释放主动技能
```

::: tip 具体按键
不同服务器配置可能略有不同，进服后留意 HUD 或公告提示的技能释放方式。
:::

## 技能跟着武器走

技能和武器流派绑定：

```
手持剑 → 能放剑系技能（旋风斩等）
换成弓 → 能放弓系技能（穿透箭等）
        剑系技能这时放不了
```

但你**已经学会的技能不会丢**。换回剑，剑系技能立刻又能用了。

## 冷却机制

- 每个技能有自己的冷却时间。
- 冷却**独立计算**，换武器不会重置（防止刷技能）。
- HUD 会显示技能还有多久好。

## 技能点怎么分配

技能点有限，要想清楚怎么花：

```
专精打法：把点集中投一个流派，技能全开
分散打法：多个流派各投一些，样样能用
```

::: tip 不用怕点错
技能点是跟着熟练度等级源源不断给的，多练几级就有更多点。不必纠结一时的分配。
:::

## 技能搭配出奇效

技能之间、技能和武器词条之间能搭配出强力 combo：

```
例：剑「旋风斩」（范围攻击）+ 武器词条「灼烧」
   → 旋风斩扫到的怪全部被点燃
   → 范围灼烧，一片怪一起掉血
```

更多搭配见 [搭配与 Combo](/advanced/combos)。

## 各流派技能预览

每个流派都有 5+ 个技能，从低级到高级逐步解锁。举几个例子：

| 流派 | 招牌技能 | 效果 |
|------|----------|------|
| 剑士 | 旋风斩 | 范围横扫 |
| 狂战 | 重劈 | 蓄力重击+击退 |
| 游侠 | 穿透箭 | 穿透多个敌人 |
| 法师 | 火球术 | 范围燃烧 |
| 刺客 | 斩首 | 处决残血目标 |
| 武者 | 崩拳 | 消耗连段爆发 |

完整技能等你在游戏里逐个解锁探索。
