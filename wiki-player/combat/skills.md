# 技能與釋放

每個武器流派都有自己的技能。這一頁講技能怎麼學、怎麼放。

## 技能從哪來

```
升級 → 獲得技能點 → 投入當前流派技能樹 → 解鎖技能
```

技能點是升級給的，花技能點解鎖你想要的技能。

## 技能分三種

| 類型 | 怎麼生效 | 例子 |
|------|----------|------|
| 主動技能 | 你手動釋放，有冷卻 | 旋風斬、火球術 |
| 被動技能 | 滿足條件自動觸發 | 暴擊回血、低血減傷 |
| 強化天賦 | 直接加屬性 | 攻擊力+5%、暴擊率+10% |

## 主動技能怎麼放

> 默認用 **潛行 + 交互** 釋放。

```
潛行（按住 Shift）+ 右鍵 → 釋放主動技能
```

::: tip 具體按鍵
不同服務器配置可能略有不同，進服後留意 HUD 或公告提示的技能釋放方式。
:::

## 技能跟著武器走

技能和武器流派綁定：

```
手持劍 → 能放劍系技能（旋風斬等）
換成弓 → 能放弓系技能（穿透箭等）
        劍系技能這時放不了
```

但你**已經學會的技能不會丟**。換回劍，劍系技能立刻又能用了。

## 冷卻機制

- 每個技能有自己的冷卻時間。
- 冷卻**獨立計算**，換武器不會重置（防止刷技能）。
- HUD 會顯示技能還有多久好。

## 技能點怎麼分配

技能點有限，要想清楚怎麼花：

```
專精打法：把點集中投一個流派，技能全開
分散打法：多個流派各投一些，樣樣能用
```

::: tip 不用怕點錯
技能點是跟著熟練度等級源源不斷給的，多練幾級就有更多點。不必糾結一時的分配。
:::

## 技能搭配出奇效

技能之間、技能和武器詞條之間能搭配出強力 combo：

```
例：劍「旋風斬」（範圍攻擊）+ 武器詞條「灼燒」
   → 旋風斬掃到的怪全部被點燃
   → 範圍灼燒，一片怪一起掉血
```

更多搭配見 [搭配與 Combo](/advanced/combos)。

## 各流派技能預覽

每個流派都有 5+ 個技能，從低級到高級逐步解鎖。舉幾個例子：

| 流派 | 招牌技能 | 效果 |
|------|----------|------|
| 劍士 | 旋風斬 | 範圍橫掃 |
| 狂戰 | 重劈 | 蓄力重擊+擊退 |
| 遊俠 | 穿透箭 | 穿透多個敵人 |
| 法師 | 火球術 | 範圍燃燒 |
| 刺客 | 斬首 | 處決殘血目標 |
| 武者 | 崩拳 | 消耗連段爆發 |

完整技能等你在遊戲裡逐個解鎖探索。
