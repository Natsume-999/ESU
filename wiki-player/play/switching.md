# 切換玩法

界限 的精髓在於"自由"。這一頁講清楚怎麼切換、切換有什麼講究。

## 怎麼切換

簡單到不能再簡單：**換主手武器就行**。

```
切換快捷欄到弓 → 立刻變成遊俠
切回劍         → 立刻變回劍士
```

沒有確認、沒有讀條、沒有冷卻。切換是瞬間的。

## 切換零代價

| 你擔心的 | 實際情況 |
|----------|----------|
| 要花錢轉職？ | 不要，免費 |
| 有冷卻時間？ | 沒有，隨時切 |
| 會丟進度？ | 不會，熟練度都在 |
| 戰鬥中能切？ | 能，隨時切 |

## 切換時發生什麼

```
你換武器
  ↓
舊職業的屬性加成移除
  ↓
新武器識別為新職業
  ↓
套上新職業的屬性 + 技能
  ↓
HUD 刷新，顯示新職業
```

一瞬間完成，你幾乎感覺不到延遲。

## 有什麼講究嗎？

雖然切換自由，但有個小機制要知道：

::: tip 技能冷卻獨立記
每個職業的技能冷卻**獨立計算**，而且切換武器不會重置。

舉例：你用劍放了「旋風斬」（進入冷卻），換成弓再換回劍——旋風斬還在冷卻中，不會因為換武器就刷新。

這是為了公平，防止靠換武器刷技能。
:::

## 實戰中的切換技巧

聰明的玩家會利用切換應對不同情況：

```
遠處一群怪 → 拿弓/法杖遠程清
怪衝到臉上 → 換劍/斧近戰砍
想跑路     → 換匕首/徒手，跑得快
```

會切換的玩家，永遠能用最合適的武器應對當前局面。

## 專精還是全能？

切換自由意味著你可以選擇自己的風格：

| 風格 | 怎麼玩 | 適合誰 |
|------|--------|--------|
| 專精 | 主練 1 種武器，練到極致 | 喜歡鑽研一種玩法 |
| 雙修 | 練 2 種互補武器（近戰+遠程） | 喜歡靈活應變 |
| 全能 | 啥都練一點 | 喜歡樣樣都會 |

沒有對錯，看你喜歡。系統都支持。
