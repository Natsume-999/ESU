# 切换玩法

ESU 的精髓在于"自由"。这一页讲清楚怎么切换、切换有什么讲究。

## 怎么切换

简单到不能再简单：**换主手武器就行**。

```
切换快捷栏到弓 → 立刻变成游侠
切回剑         → 立刻变回剑士
```

没有确认、没有读条、没有冷却。切换是瞬间的。

## 切换零代价

| 你担心的 | 实际情况 |
|----------|----------|
| 要花钱转职？ | 不要，免费 |
| 有冷却时间？ | 没有，随时切 |
| 会丢进度？ | 不会，熟练度都在 |
| 战斗中能切？ | 能，随时切 |

## 切换时发生什么

```
你换武器
  ↓
旧职业的属性加成移除
  ↓
新武器识别为新职业
  ↓
套上新职业的属性 + 技能
  ↓
HUD 刷新，显示新职业
```

一瞬间完成，你几乎感觉不到延迟。

## 有什么讲究吗？

虽然切换自由，但有个小机制要知道：

::: tip 技能冷却独立记
每个职业的技能冷却**独立计算**，而且切换武器不会重置。

举例：你用剑放了「旋风斩」（进入冷却），换成弓再换回剑——旋风斩还在冷却中，不会因为换武器就刷新。

这是为了公平，防止靠换武器刷技能。
:::

## 实战中的切换技巧

聪明的玩家会利用切换应对不同情况：

```
远处一群怪 → 拿弓/法杖远程清
怪冲到脸上 → 换剑/斧近战砍
想跑路     → 换匕首/徒手，跑得快
```

会切换的玩家，永远能用最合适的武器应对当前局面。

## 专精还是全能？

切换自由意味着你可以选择自己的风格：

| 风格 | 怎么玩 | 适合谁 |
|------|--------|--------|
| 专精 | 主练 1 种武器，练到极致 | 喜欢钻研一种玩法 |
| 双修 | 练 2 种互补武器（近战+远程） | 喜欢灵活应变 |
| 全能 | 啥都练一点 | 喜欢样样都会 |

没有对错，看你喜欢。系统都支持。
