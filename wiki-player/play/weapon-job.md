# 武器即职业

这是 ESU 最核心的玩法。搞懂它，你就懂了这个服务器。

## 核心规则

> 你**主手**拿着什么武器，你就是什么职业。

```
主手持剑   → 剑士（连击、暴击）
主手持斧   → 狂战（高伤、破甲）
主手持弓   → 游侠（远程输出）
主手持法杖 → 法师（范围法术）
主手持匕首 → 刺客（背刺、机动）
空手       → 武者（徒手格斗）
```

换武器的瞬间，你的属性、技能、战斗风格全部切换。

## 每种职业玩起来不一样

不只是数值不同，**玩法手感**也不同：

| 职业 | 武器 | 玩起来什么感觉 |
|------|------|----------------|
| 剑士 | 剑 | 流畅连击，攻守平衡，新手友好 |
| 狂战 | 斧 | 慢但狠，一下打很疼，无视护甲 |
| 游侠 | 弓 | 拉开距离风筝，越远打越疼 |
| 法师 | 法杖 | 扔范围法术，烧一片冻一片 |
| 刺客 | 匕首 | 又快又灵，绕背暴击，秒残血 |
| 武者 | 徒手 | 赤手连段，跑得快，灵活 |

## 武器决定什么

手持武器决定三样东西：

```
        当前武器
           │
   ┌───────┼───────┐
   ▼       ▼       ▼
 属性    技能    战斗风格
(攻防等) (能放啥)  (怎么打)
```

切换武器，这三样同时切换。

## 同一种武器有强弱

同一个流派里，武器有好坏之分：

```
剑流派：木剑 < 铁剑 < 钻石剑 < 传说之剑
        （都是剑，都用剑系技能和熟练度）
        （但好剑属性更高、可能带特殊效果）
```

换更好的同类武器，你的熟练度和技能**完全继承**，因为它们记在你身上，不记在武器上。

## 为什么这样设计

这套设计让你：

- **不怕选错**：随时能换，没有沉没成本。
- **想专精就专精**：一直用剑，成为剑术大师。
- **想全能就全能**：剑弓法杖都练，应对各种情况。
- **一个号玩够本**：不用开小号体验不同玩法。
