# 武器即職業

這是 界限 最核心的玩法。搞懂它，你就懂了這個服務器。

## 核心規則

> 你**主手**拿著什麼武器，你就是什麼職業。

```
主手持劍   → 劍士（連擊、暴擊）
主手持斧   → 狂戰（高傷、破甲）
主手持弓   → 遊俠（遠程輸出）
主手持法杖 → 法師（範圍法術）
主手持匕首 → 刺客（背刺、機動）
空手       → 武者（徒手格鬥）
```

換武器的瞬間，你的屬性、技能、戰鬥風格全部切換。

## 每種職業玩起來不一樣

不只是數值不同，**玩法手感**也不同：

| 職業 | 武器 | 玩起來什麼感覺 |
|------|------|----------------|
| 劍士 | 劍 | 流暢連擊，攻守平衡，新手友好 |
| 狂戰 | 斧 | 慢但狠，一下打很疼，無視護甲 |
| 遊俠 | 弓 | 拉開距離風箏，越遠打越疼 |
| 法師 | 法杖 | 扔範圍法術，燒一片凍一片 |
| 刺客 | 匕首 | 又快又靈，繞背暴擊，秒殘血 |
| 武者 | 徒手 | 赤手連段，跑得快，靈活 |

## 武器決定什麼

手持武器決定三樣東西：

```
        當前武器
           │
   ┌───────┼───────┐
   ▼       ▼       ▼
 屬性    技能    戰鬥風格
(攻防等) (能放啥)  (怎麼打)
```

切換武器，這三樣同時切換。

## 同一種武器有強弱

同一個流派裡，武器有好壞之分：

```
劍流派：木劍 < 鐵劍 < 鑽石劍 < 傳說之劍
        （都是劍，都用劍系技能和熟練度）
        （但好劍屬性更高、可能帶特殊效果）
```

換更好的同類武器，你的熟練度和技能**完全繼承**，因為它們記在你身上，不記在武器上。

## 為什麼這樣設計

這套設計讓你：

- **不怕選錯**：隨時能換，沒有沉沒成本。
- **想專精就專精**：一直用劍，成為劍術大師。
- **想全能就全能**：劍弓法杖都練，應對各種情況。
- **一個號玩夠本**：不用開小號體驗不同玩法。
