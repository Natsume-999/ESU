# 熟练度成长

熟练度是你变强的核心。这一页讲清楚怎么练、怎么练得快。

## 核心规则

> 用某把武器战斗，就积累**那把武器**的熟练度。

```
用剑打怪 → 剑熟练度 +经验
换弓打怪 → 弓熟练度 +经验（剑不变）
```

**用什么练什么**。想练哪个武器，就用哪个武器战斗。

## 熟练度怎么涨

主要靠**击杀怪物**：

```
击杀怪物 → 按怪物血量给经验 → 当前武器熟练度增加
```

怪物越强（血量越高），给的经验越多。

## 升级带来什么

熟练度满了就升级，每次升级你会获得：

| 回报 | 说明 |
|------|------|
| 属性提升 | 攻击更高、更耐打等 |
| 技能点 | 用来解锁/强化技能 |
| 解锁技能 | 到特定等级解锁新技能 |

::: tip 属性跟着武器走
你的剑系属性只在**手持剑**时生效。换成弓，就变成弓系属性。每个武器流派的属性是分开的。
:::

## 多个武器一起练

你可以同时练好几个武器，各自独立：

```
你的熟练度：
  剑   Lv 42  ████████░░
  弓   Lv 15  ███░░░░░░░
  法杖 Lv 3   █░░░░░░░░░
```

今天想练剑就用剑，明天想练弓就用弓。每条线独立涨，互不影响。

## 怎么练得快

### ✅ 有效的做法

- 打**和你等级相当或更强**的怪，经验多。
- 持续战斗，保持活跃。
- 挑战副本、Boss 等高价值目标。

### ❌ 没用的做法

::: warning 这些练不动
- **挂机刷怪笼**：系统判定挂机，经验大砍。
- **虐杀超低级怪**：等级压制，经验很少。
- **站着不动让怪撞**：判定挂机，几乎没经验。

熟练度奖励的是**真正的战斗**，不是挂机。
:::

## 熟练度 vs 装备

很多新人问：是练熟练度重要，还是刷好装备重要？

> **熟练度是根本，装备是加分。**

一个高熟练度玩家用普通武器，照样打得过低熟练度玩家用神装。所以——**先踏实练熟练度**，好装备是锦上添花。

## 长期目标

熟练度给你长线追求：

```
短期：升到下一级，解锁下一个技能
中期：把一个武器练到精通
长期：练满所有武器流派，成为全能高手
```
