# 熟練度成長

熟練度是你變強的核心。這一頁講清楚怎麼練、怎麼練得快。

## 核心規則

> 用某把武器戰鬥，就積累**那把武器**的熟練度。

```
用劍打怪 → 劍熟練度 +經驗
換弓打怪 → 弓熟練度 +經驗（劍不變）
```

**用什麼練什麼**。想練哪個武器，就用哪個武器戰鬥。

## 熟練度怎麼漲

主要靠**擊殺怪物**：

```
擊殺怪物 → 按怪物血量給經驗 → 當前武器熟練度增加
```

怪物越強（血量越高），給的經驗越多。

## 升級帶來什麼

熟練度滿了就升級，每次升級你會獲得：

| 回報 | 說明 |
|------|------|
| 屬性提升 | 攻擊更高、更耐打等 |
| 技能點 | 用來解鎖/強化技能 |
| 解鎖技能 | 到特定等級解鎖新技能 |

::: tip 屬性跟著武器走
你的劍系屬性只在**手持劍**時生效。換成弓，就變成弓系屬性。每個武器流派的屬性是分開的。
:::

## 多個武器一起練

你可以同時練好幾個武器，各自獨立：

```
你的熟練度：
  劍   Lv 42  ████████░░
  弓   Lv 15  ███░░░░░░░
  法杖 Lv 3   █░░░░░░░░░
```

今天想練劍就用劍，明天想練弓就用弓。每條線獨立漲，互不影響。

## 怎麼練得快

### ✅ 有效的做法

- 打**和你等級相當或更強**的怪，經驗多。
- 持續戰鬥，保持活躍。
- 挑戰副本、Boss 等高價值目標。

### ❌ 沒用的做法

::: warning 這些練不動
- **掛機刷怪籠**：系統判定掛機，經驗大砍。
- **虐殺超低級怪**：等級壓制，經驗很少。
- **站著不動讓怪撞**：判定掛機，幾乎沒經驗。

熟練度獎勵的是**真正的戰鬥**，不是掛機。
:::

## 熟練度 vs 裝備

很多新人問：是練熟練度重要，還是刷好裝備重要？

> **熟練度是根本，裝備是加分。**

一個高熟練度玩家用普通武器，照樣打得過低熟練度玩家用神裝。所以——**先踏實練熟練度**，好裝備是錦上添花。

## 長期目標

熟練度給你長線追求：

```
短期：升到下一級，解鎖下一個技能
中期：把一個武器練到精通
長期：練滿所有武器流派，成為全能高手
```
