# 搭配與 Combo

高手和新手的差距，往往在於搭配。這一頁教你把技能、武器詞條組合出強力 combo。

## Combo 的兩種來源

```
技能 + 技能       → 技能連招
技能 + 武器詞條   → 效果聯動
```

會搭配的玩家，傷害和生存都遠超只會單發技能的玩家。

## 經典 Combo 示例

### 範圍灼燒（劍士）

```
劍「旋風斬」（範圍橫掃） + 武器詞條「灼燒」
  ↓
旋風斬掃中的所有怪 → 全部被點燃
  ↓
一群怪同時持續掉血
```

適合清理怪群。

### 冰火兩重天（法師）

```
法杖「冰霜新星」（冰凍定身） → 法杖「火球術」（範圍爆發）
  ↓
先凍住一群怪 → 再火球轟炸
  ↓
怪跑不掉，全吃滿傷害
```

控制 + 爆發的經典連招。

### 處決連（刺客）

```
匕首「影襲」（多段攻擊打殘） → 匕首「斬首」（處決殘血）
  ↓
影襲把怪打到低血 → 斬首直接處決
  ↓
快速擊殺單體
```

專殺高血量精英。

### 蓄力狙殺（遊俠）

```
被動「鷹眼」（距離越遠越疼） + 「貫日之箭」（蓄力重擊）
  ↓
拉到最遠距離 → 蓄滿力射出
  ↓
極限單體爆發傷害
```

遠程秒殺流。

### 連段爆發（武者）

```
「連環拳」（累積連段層數） → 「崩拳」（消耗連段爆發）
  ↓
先連擊疊滿層數 → 崩拳一次性釋放
  ↓
層數越高，崩拳越疼
```

徒手也能打出高爆發。

## 武器詞條聯動思路

不同詞條配合不同打法：

| 詞條 | 配合什麼 |
|------|----------|
| 吸血 | 配高頻攻擊（劍、匕首），邊打邊回血 |
| 灼燒 | 配範圍技能，一次點燃一片 |
| 雷擊（暴擊觸發） | 配高暴擊流派（刺客），頻繁觸發連鎖 |
| 穿甲 | 配高傷流派（狂戰），打肉盾更疼 |

## 跨流派思路（進階）

雖然技能不能跨流派同時放，但你可以**切換**著用：

```
法師冰凍一群怪 → 切劍衝進去旋風斬
（先用一個流派控制，切另一個流派輸出）
```

這需要練多個流派，是老手的高階玩法。

## 怎麼開發自己的 Combo

```
1. 看你的技能：哪些能控制、哪些能爆發、哪些是範圍
2. 看你的武器詞條：觸發條件是什麼
3. 想順序：控制→爆發，或鋪狀態→引爆
4. 實戰測試，調整
```

::: tip Combo 是 界限 的深度
單發技能誰都會，但把技能、詞條、切換組合起來，才是 界限 戰鬥的精髓。多試驗，找到屬於你的連招。
:::
