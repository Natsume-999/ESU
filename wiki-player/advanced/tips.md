# 成長技巧

想更快變強？這些技巧幫你高效成長。

## 高效練熟練度

### 打對怪

```
✅ 打和你等級相當或更強的怪 → 經驗多
❌ 虐殺遠低於你的怪 → 等級壓制，經驗少
```

找適合你當前等級的區域刷怪，效率最高。

### 保持活躍戰鬥

```
✅ 連續移動、戰鬥 → 滿經驗
❌ 站樁掛機、刷怪籠 → 判定掛機，經驗大砍
```

系統獎勵真正的戰鬥，別想著掛機偷懶。

### 挑戰高價值目標

副本、Boss、精英怪給的經驗遠超普通小怪。能挑戰就別隻刷小怪。

## 專精 vs 全能怎麼選

### 專精路線（推薦新手）

先把**一個**武器練起來：

```
好處：
- 早早擁有一套強力技能
- 戰鬥力集中，打得動更強的怪
- 不分散，成長快
```

建議新手先專精一個順手的流派，到中期再考慮練第二個。

### 全能路線（適合老手）

練多個武器流派：

```
好處：
- 應對各種場景（遠近、單體群體）
- 玩法豐富不膩
壞處：
- 前期分散，單個流派成長慢
```

::: tip 我的建議
**新手先專精，老手再全能。** 先用一個流派站穩腳跟，有了戰鬥力，再慢慢拓展其它流派會輕鬆很多。
:::

## 善用切換

就算你專精一個流派，也建議**副練一個互補的**：

```
主練近戰（劍/斧）+ 副練遠程（弓）
→ 怪遠了用弓拉，近了用劍砍
→ 應對更靈活
```

## 武器選擇

```
1. 熟練度優先：先練熟練度，再追裝備
2. 同類升級：換更好的同類武器，熟練度繼承，無縫變強
3. 看詞條：高級武器的特殊詞條能大幅提升戰力
```

## 技能點分配

```
專精期：集中投一個流派，快速成型
拓展期：新流派從核心技能開始投
```

別把點撒得太散，先保證主流派的關鍵技能到位。

## 成長節奏參考

| 階段 | 目標 |
|------|------|
| 新手 | 專精一個流派，解鎖前幾個技能 |
| 成長 | 主流派練到中高級，副流派起步 |
| 成熟 | 主流派精通，多流派發展 |
| 老手 | 追求練滿、收集稀有武器、挑戰終局 |
