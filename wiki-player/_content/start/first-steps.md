# 第一次游玩

跟着这几步，开启你的 界限 之旅。

## 第一步：进服

进入服务器后，你**不需要**选职业，也没有冗长的新手问卷。系统会自动为你创建角色，你直接就能开始。

## 第二步：拿起武器

你会拿到一把基础武器（比如新手剑）。把它握在主手上——

```
主手持剑 → 屏幕 HUD 显示「剑士」 → 你现在就是剑士了
```

就这么简单。手里的武器决定了你是什么职业。

## 第三步：开打

去打怪吧。每次有效战斗（击杀怪物）都会给你**当前武器**积累熟练度。

```
用剑打怪 → 剑的熟练度增加 → 进度条往前走
```

HUD 上会显示你的熟练度进度条，你能直观看到自己在变强。

## 第四步：第一次升级

打着打着，熟练度满了，你会**升级**：

- 属性提升（攻击更疼、更耐打）
- 获得技能点
- 可能解锁新技能

升级时会有提示，你能明显感觉到"我变强了"。

## 第五步：试试别的武器

拿到第二把武器（比如弓）？换上试试——

```
换上弓 → HUD 变成「游侠」 → 玩法完全不同
```

注意：弓的熟练度是**独立**的，从头开始练。但你的剑熟练度不会丢，随时换回去都在。

## 你已经会玩了

到这里，你已经掌握了 界限 的核心：

- ✅ 拿武器 = 选职业
- ✅ 战斗 = 变强
- ✅ 换武器 = 换玩法，零代价

接下来就是探索这个世界，练你喜欢的武器，变得更强。
