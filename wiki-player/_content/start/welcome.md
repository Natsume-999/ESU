# 欢迎来到 界限

欢迎！这是一个和别处不太一样的 RPG 服务器。三分钟，带你搞懂它怎么玩。

## 一句话规则

> **你拿什么武器，你就是什么职业。**

不用在创建角色时纠结选哪个职业。进服，拿起武器，开打。就这么简单。

## 和别的服务器哪里不一样？

| 别的服务器 | 界限 |
|------------|-----|
| 开局选职业，选错难改 | 不用选，拿武器即职业 |
| 转职要花钱/有代价 | 换武器就换职业，免费 |
| 一个号一种玩法 | 一个号玩遍所有武器 |
| 练废了要重开 | 随时换，永远不亏 |

## 你会怎么玩

```
1. 进服 → 直接有武器，直接开打
2. 用武器战斗 → 这把武器越来越强
3. 想换花样 → 换把武器，变成另一个职业
4. 慢慢地 → 你练熟了好几种武器，想用哪个用哪个
```

## 三个核心概念

### 武器即职业

手里的武器决定你的战斗风格、属性和技能。换武器 = 换职业。

👉 详见 [武器即职业](/play/weapon-job)

### 熟练度成长

用某把武器战斗，就积累它的熟练度。熟练度越高，这把武器越强、技能越多。

👉 详见 [熟练度成长](/play/mastery)

### 自由切换

任何时候都能换武器换玩法，没有代价。想专精就一直用一种，想全能就轮着练。

👉 详见 [切换玩法](/play/switching)
