# 搭配与 Combo

高手和新手的差距，往往在于搭配。这一页教你把技能、武器词条组合出强力 combo。

## Combo 的两种来源

```
技能 + 技能       → 技能连招
技能 + 武器词条   → 效果联动
```

会搭配的玩家，伤害和生存都远超只会单发技能的玩家。

## 经典 Combo 示例

### 范围灼烧（剑士）

```
剑「旋风斩」（范围横扫） + 武器词条「灼烧」
  ↓
旋风斩扫中的所有怪 → 全部被点燃
  ↓
一群怪同时持续掉血
```

适合清理怪群。

### 冰火两重天（法师）

```
法杖「冰霜新星」（冰冻定身） → 法杖「火球术」（范围爆发）
  ↓
先冻住一群怪 → 再火球轰炸
  ↓
怪跑不掉，全吃满伤害
```

控制 + 爆发的经典连招。

### 处决连（刺客）

```
匕首「影袭」（多段攻击打残） → 匕首「斩首」（处决残血）
  ↓
影袭把怪打到低血 → 斩首直接处决
  ↓
快速击杀单体
```

专杀高血量精英。

### 蓄力狙杀（游侠）

```
被动「鹰眼」（距离越远越疼） + 「贯日之箭」（蓄力重击）
  ↓
拉到最远距离 → 蓄满力射出
  ↓
极限单体爆发伤害
```

远程秒杀流。

### 连段爆发（武者）

```
「连环拳」（累积连段层数） → 「崩拳」（消耗连段爆发）
  ↓
先连击叠满层数 → 崩拳一次性释放
  ↓
层数越高，崩拳越疼
```

徒手也能打出高爆发。

## 武器词条联动思路

不同词条配合不同打法：

| 词条 | 配合什么 |
|------|----------|
| 吸血 | 配高频攻击（剑、匕首），边打边回血 |
| 灼烧 | 配范围技能，一次点燃一片 |
| 雷击（暴击触发） | 配高暴击流派（刺客），频繁触发连锁 |
| 穿甲 | 配高伤流派（狂战），打肉盾更疼 |

## 跨流派思路（进阶）

虽然技能不能跨流派同时放，但你可以**切换**着用：

```
法师冰冻一群怪 → 切剑冲进去旋风斩
（先用一个流派控制，切另一个流派输出）
```

这需要练多个流派，是老手的高阶玩法。

## 怎么开发自己的 Combo

```
1. 看你的技能：哪些能控制、哪些能爆发、哪些是范围
2. 看你的武器词条：触发条件是什么
3. 想顺序：控制→爆发，或铺状态→引爆
4. 实战测试，调整
```

::: tip Combo 是 界限 的深度
单发技能谁都会，但把技能、词条、切换组合起来，才是 界限 战斗的精髓。多试验，找到属于你的连招。
:::
