# 成长技巧

想更快变强？这些技巧帮你高效成长。

## 高效练熟练度

### 打对怪

```
✅ 打和你等级相当或更强的怪 → 经验多
❌ 虐杀远低于你的怪 → 等级压制，经验少
```

找适合你当前等级的区域刷怪，效率最高。

### 保持活跃战斗

```
✅ 连续移动、战斗 → 满经验
❌ 站桩挂机、刷怪笼 → 判定挂机，经验大砍
```

系统奖励真正的战斗，别想着挂机偷懒。

### 挑战高价值目标

副本、Boss、精英怪给的经验远超普通小怪。能挑战就别只刷小怪。

## 专精 vs 全能怎么选

### 专精路线（推荐新手）

先把**一个**武器练起来：

```
好处：
- 早早拥有一套强力技能
- 战斗力集中，打得动更强的怪
- 不分散，成长快
```

建议新手先专精一个顺手的流派，到中期再考虑练第二个。

### 全能路线（适合老手）

练多个武器流派：

```
好处：
- 应对各种场景（远近、单体群体）
- 玩法丰富不腻
坏处：
- 前期分散，单个流派成长慢
```

::: tip 我的建议
**新手先专精，老手再全能。** 先用一个流派站稳脚跟，有了战斗力，再慢慢拓展其它流派会轻松很多。
:::

## 善用切换

就算你专精一个流派，也建议**副练一个互补的**：

```
主练近战（剑/斧）+ 副练远程（弓）
→ 怪远了用弓拉，近了用剑砍
→ 应对更灵活
```

## 武器选择

```
1. 熟练度优先：先练熟练度，再追装备
2. 同类升级：换更好的同类武器，熟练度继承，无缝变强
3. 看词条：高级武器的特殊词条能大幅提升战力
```

## 技能点分配

```
专精期：集中投一个流派，快速成型
拓展期：新流派从核心技能开始投
```

别把点撒得太散，先保证主流派的关键技能到位。

## 成长节奏参考

| 阶段 | 目标 |
|------|------|
| 新手 | 专精一个流派，解锁前几个技能 |
| 成长 | 主流派练到中高级，副流派起步 |
| 成熟 | 主流派精通，多流派发展 |
| 老手 | 追求练满、收集稀有武器、挑战终局 |
