# 第一次遊玩

跟著這幾步，開啟你的 界限 之旅。

## 第一步：進服

進入服務器後，你**不需要**選職業，也沒有冗長的新手問卷。系統會自動為你創建角色，你直接就能開始。

## 第二步：拿起武器

你會拿到一把基礎武器（比如新手劍）。把它握在主手上——

```
主手持劍 → 屏幕 HUD 顯示「劍士」 → 你現在就是劍士了
```

就這麼簡單。手裡的武器決定了你是什麼職業。

## 第三步：開打

去打怪吧。每次有效戰鬥（擊殺怪物）都會給你**當前武器**積累熟練度。

```
用劍打怪 → 劍的熟練度增加 → 進度條往前走
```

HUD 上會顯示你的熟練度進度條，你能直觀看到自己在變強。

## 第四步：第一次升級

打著打著，熟練度滿了，你會**升級**：

- 屬性提升（攻擊更疼、更耐打）
- 獲得技能點
- 可能解鎖新技能

升級時會有提示，你能明顯感覺到"我變強了"。

## 第五步：試試別的武器

拿到第二把武器（比如弓）？換上試試——

```
換上弓 → HUD 變成「遊俠」 → 玩法完全不同
```

注意：弓的熟練度是**獨立**的，從頭開始練。但你的劍熟練度不會丟，隨時換回去都在。

## 你已經會玩了

到這裡，你已經掌握了 界限 的核心：

- ✅ 拿武器 = 選職業
- ✅ 戰鬥 = 變強
- ✅ 換武器 = 換玩法，零代價

接下來就是探索這個世界，練你喜歡的武器，變得更強。
