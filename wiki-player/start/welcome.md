# 歡迎來到 界限

歡迎！這是一個和別處不太一樣的 RPG 服務器。三分鐘，帶你搞懂它怎麼玩。

## 一句話規則

> **你拿什麼武器，你就是什麼職業。**

不用在創建角色時糾結選哪個職業。進服，拿起武器，開打。就這麼簡單。

## 和別的服務器哪裡不一樣？

| 別的服務器 | 界限 |
|------------|-----|
| 開局選職業，選錯難改 | 不用選，拿武器即職業 |
| 轉職要花錢/有代價 | 換武器就換職業，免費 |
| 一個號一種玩法 | 一個號玩遍所有武器 |
| 練廢了要重開 | 隨時換，永遠不虧 |

## 你會怎麼玩

```
1. 進服 → 直接有武器，直接開打
2. 用武器戰鬥 → 這把武器越來越強
3. 想換花樣 → 換把武器，變成另一個職業
4. 慢慢地 → 你練熟了好幾種武器，想用哪個用哪個
```

## 三個核心概念

### 武器即職業

手裡的武器決定你的戰鬥風格、屬性和技能。換武器 = 換職業。

👉 詳見 [武器即職業](/play/weapon-job)

### 熟練度成長

用某把武器戰鬥，就積累它的熟練度。熟練度越高，這把武器越強、技能越多。

👉 詳見 [熟練度成長](/play/mastery)

### 自由切換

任何時候都能換武器換玩法，沒有代價。想專精就一直用一種，想全能就輪著練。

👉 詳見 [切換玩法](/play/switching)
